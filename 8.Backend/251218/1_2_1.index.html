2단계: CSS - 스타일과 레이아웃 (살붙이기)

문서를 시각적으로 구성하고 배치하는 방법을 배웁니다.

선택자 (Selectors)

선택자는 CSS 규칙을 어떤 요소에 적용할지 결정합니다.

1. 태그 선택자

모든 <p> 태그에 적용됩니다. 이 단락도 포함됩니다.

이것도 태그 선택자의 영향을 받습니다.

2. 클래스 선택자 (.)

3. 아이디 선택자 (#)

아이디는 페이지에서 고유해야 합니다. 주로 특정 요소를 JavaScript로 조작할 때 사용됩니다.

우선순위 (Specificity): 아이디 > 클래스 > 태그

박스 모델 (Box Model)

모든 HTML 요소는 박스 모델로 이루어져 있습니다: content → padding → border → margin

Content
(300x150)

박스 모델 구성 요소

  • Content: 실제 내용이 들어가는 영역
  • Padding: 내용과 테두리 사이의 안쪽 공간
  • Border: 테두리 (스타일, 색상, 굵기 지정 가능)
  • Margin: 테두리 바깥쪽 공간 (다른 요소와의 거리)

Box-sizing 팁

box-sizing: border-box;를 사용하면 padding과 border가 width에 포함됩니다. 매우 유용합니다!

* {
    box-sizing: border-box;
}

포지셔닝 (Positioning)

1. Static (기본값)

이것은 static 포지셔닝입니다. 일반적인 문서 흐름을 따릅니다.

2. Relative

이것은 relative 포지셔닝입니다. 원래 위치를 기준으로 이동합니다. left: 20px로 오른쪽으로 20px 이동했습니다.

3. Absolute

부모 컨테이너 (position: relative)
Absolute 자식

자식 요소가 부모 요소 기준으로 절대 위치로 배치됩니다.

4. Fixed

우측 하단에 고정된 박스를 확인할 수 있습니다.

실무 팁: Absolute와 Fixed는 문서 흐름을 벗어나므로 조심히 사용해야 합니다.

Flexbox - 현대적 레이아웃 (실무에서 가장 많이 사용)

Flex 컨테이너와 아이템

display: flex;
justify-content: center;
align-items: center;
gap: 15px;
Flexbox 장점: 복잡한 레이아웃을 간단하게 만들 수 있습니다.

반응형 디자인 (Responsive Design)

미디어 쿼리를 사용하여 화면 크기에 따라 다른 스타일을 적용합니다.

반응형 디자인의 핵심:
  • 모바일 우선 설계 (Mobile-First)
  • 유연한 레이아웃 (Flexbox, Grid)
  • 미디어 쿼리로 화면 크기별 조정
